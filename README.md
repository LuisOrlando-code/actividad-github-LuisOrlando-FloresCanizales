Objectivo:
Que el alumno practique el flujo completo de trabajo en GitHub partiendo desde cero: crear un repositorio nuevo, clonarlo, trabajar en una rama, realizar cambios y generar un Pull Request (PR).

Qué es un Pull Request (definición breve):
Un Pull Request (PR) o solicitud de extracción es una funcionalidad de plataformas como GitHub o Bitbucket que permite a los desarrolladores notificar a su equipo sobre cambios realizados en una rama de un repositorio. Sirve para revisar, discutir y fusionar (merge) dicho código con la rama principal, garantizando la calidad antes de su integración final.
